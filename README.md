# qim-watermark-python
Projet de tatouage numérique d'image utilisant la méthode QIM (DCT + PSNR)
# 🧠 Tatouage Numérique d’Images (QIM)

## 📌 Description
Ce projet implémente une technique de tatouage numérique basée sur :
- DCT (Discrete Cosine Transform)
- QIM (Quantization Index Modulation)

Permet :
- D'insérer un watermark (bits) dans une image
- D'extraire ce watermark
- D'évaluer la qualité avec PSNR

## 🛠️ Technologies
- Python
- OpenCV
- NumPy
- Matplotlib
- Tkinter
